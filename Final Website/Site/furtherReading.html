Extra Reading

Below are links to other websites and resources on the web. These are all useful to a website developer and include: further tutorial sites, code validation sites and other general useful sites to a website developer. These extra sources can offer more in depth tutorials than the ones covered on this site, so are very useful to anyone who is looking to progress their skills as a web site developer or if you want to learn some new tricks.

w3schools.com

w3schools has almost everything you could ever need for making a website. It is filled with tutorials on how to use HTML, CSS and JavaScript. It not only covers the basics like this site does, but also offers more complex and in depth tutorials. It also has information on everything website related, such as lists of attributes you can use as well as how and when to use them. Click the image to go and see what they have to offer.

LearnWebCode.com

LearnWebCode is another site filled with tutorials for creating websites. It goes right frome the basics of making sites to advanced concepts. It also has a blog with new concepts and ideas to try. This site is great for beginers, as it offers YouTube videos to give visual tutorials with commentary, walking a new web site creator through each part step by step. You can find these tutorials and more by clicking on the image.

Codecademy.com

Codecademy is a site wholey dedicated to giving you courses which contain step by step tutorials to teach you everything that you need to know about a programming language. While this site covers a range of programming languages, it also has a HTML & CSS course. This course will walk you through each part, from the basics to projects you can try. It is all automatic so that you can know immediatley if you are doing the right thing. Click the image to go and see their HTML & CSS course.

Validation Services

HTML and CSS both have various standardisation to them. Depending on which type you are using, you will need to ensure that the code which you write is valid within the standard that you are writing to. As different standardisations have some attributes which are available to them, but which are not to others. To ensure that your code is valid for that type, you can use w3schools' online validators. Submit your code to them and validate them to the required standard. You can find the HTML validator here and the CSS validator here.
