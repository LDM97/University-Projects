HTML

Hypertext Markup Language (HTML) is the basic framework of all websites. It uses what are known as tags to seperate different elements of the page to allow them to be accessed and formatted by CSS and JavaScript. Simple web pages can be made using HTML alone as it provides a basic level of formatting to allow headings, text, pictures, colors and more to be added to a page. Below are sections covering the basics of HTML that you need to know in order to start making web pages.

Tags

HTML is made up of tags. Tags are used to specify the different elements that make up the page. Elements are designated by placing them within the appropriate tag. Tags are within a "<" and a ">" symbol. For example, the heading tag (used to designate headings) can come in an array of sizes, with 1 being the biggest header. A header tag would look like: < h1 > this is a title < /h1 >, with the text in between the tags representing the text that will be displayed as a heading. It is important to notice that every tag has a matching closing tag, which is the tag's name within the "< >" but with a "/" before the name to show that it is a closing tag. Closing tags mark the end of the tag and its content. If tags are not closed, the rest of the page will be interpreted as being part of that tag, thus it is good practice to write the tags close when you write the opening and then add the content.

Below is a table containing some of the most common and important tags that are needed to make a site. Note, there are many more than the ones here.

Useful HTML Tags
Tag Description
< !DOCTYPE > Defines the type of document. (use < !DOCTYPE html > to define it as a html type document).
< html > Defines the HTML section of the document. All of your HTML code will go within this tag.
< title > Defines the title section of the document.
< body > Defines the body of the document. All of the code for the page will go within this tag.
< h1 > Defines the headers/titles of a document. Can be h1 - h6, decreasing in size the higher the number.
< p > Defines a paragraph. Any plain text for the page will go within this tag.
< br > Placed alone (with no closing tag) within a paragraph tag to indicate a line break at that point in the text.

Attributes

Attributes further enhance tags by allowing you to specify additional properties or behaviour for the tags that you use. This is where HTML starts to become the framework for CSS as some attributes are used to identify parts of the page so that CSS can format them. The most common are listed below. Note, there are many more than these basic ones.

Useful Tag Attributes
Attribute Description
class Associates the element with a CSS class, allowing one part of CSS code to be applied to all elements that have this class attribute.
id Uniquely identifies the given element so that it can be formatted with CSS. id is unique and two elements using the id attribute should not have the same id given to them.
align Used to specifiy the alignment for this element. Such as using align = "center" to position the element down the middle of the page.

Document Layout

An Example of Document Layout.

HTML documents follow a universal structure. The first is to define the doctype, using the < !DOCTYPE > tag. Since you are writing a HTML document, it is simpler to use < !DOCTYPE html > at the start of the document to define it as a HTML document.

Next the < html > tag is used. This tag comes after the doctype and does not close until the very end of the document. The rest of the page will be within this tag. The next is the head section, within the < head > tag. This contains information about the document to help process it. This will include the < title > tag which will define a title (that is not shown on the page). Such as: < title > My Web Page < /title >.

The document will then use a < body > tag. This is where the whole page's content will go. The body tag is closed at the end of the document, just before the html tag is closed.

Hyperlinks

The internet is built upon hyperlinks. Anything that you click on to get to another page is a hyperlink. You can create hyperlinks for your own page to link to other pages you have created or to other online content. The simplest way of doing this in HTML is to use the anchor tag. Any elements, text, images etc that are within the given anchor tag will be a hyperlink, taking you to the destination specified when clicked.

Images

Images can, of course, make a webpage more visually appealing and can break up extensive amounts of text content. Adding images to your site can be easily done using the < src > tag.

Firstly, the use of a / right before the end of the tag acts as a close for the tag, meaning that this does not require a closing tag as we have seen before. The use of the alt attribute will produce the text in quotes if the image ever fails to load. This is much cleaner to see than a broken image. The use of the height and width attributes (measured in pixels) is important when using images. If these are not specified, an image will revert to it's original resolution. If your image is very high resolution, it will cover the whole page, or if low will be difficult to see. Using the height and width tags are important to ensure that you control the images as you want to see them on your page.
