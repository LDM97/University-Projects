HTML

HTML (Hypertext Markup Language) makes up the basic frame of web pages. A basic web page can be made solely with html, but it is typically accompanied by CSS and JavaScript to make the site more visually appealing. As a result, HTML is often used as the main framework for a site's layout and main features, which are then formatted and enhanced by CSS and JavaScript. Click the image to learn the basics of how to use HTML.

CSS

CSS (Cascading Style Sheets) are used to format an existing HTML document. They work by selecting a specific tag in the HTML document and then apply formatting to the elements within that tag. CSS offers more complex formatting than HTML's basic formatting capabilities, meaning that, in good hands, CSS can be used to make a visually appealing website.
